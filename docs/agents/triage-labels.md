# Triage Labels

This repository uses the default triage vocabulary.

- `needs-triage`: maintainer needs to evaluate
- `needs-info`: waiting on reporter
- `ready-for-agent`: fully specified and ready for an AFK agent
- `ready-for-human`: needs human implementation
- `wontfix`: will not be actioned

If the issue tracker later adds custom labels, map them here instead of changing the skill assumptions.
